#BACH1 Library Documentation  

Version: 0.0.1  
Author: Deniz Erdogan  
Date: 2016-07-27

A WIP library built to create a MIDI hardware sequencer with the Arduino.

## Planned features
- MIDI sequence playback
- MIDI clock sync
- EEPROM storage (settings, patterns)
- pattern chaining
- LED control
- LCD control
- misc input control
